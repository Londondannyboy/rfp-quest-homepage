## Implementation Strategy

### 1. State-Driven Layout
```typescript
// In page.tsx
const isLoggedIn = userContext?.authenticated;

return (
  <>
    {isLoggedIn ? (
      <AppMode /> // Current chat interface
    ) : (
      <MarketingMode /> // New marketing layout
    )}
  </>
)
```

### 2. Component Architecture
```
apps/app/src/components/
├── marketing/
│   ├── hero-section.tsx         <- 21st.dev generated
│   ├── live-market-dashboard.tsx
│   ├── value-propositions.tsx   <- 21st.dev generated
│   ├── social-proof.tsx         <- 21st.dev generated
│   ├── features-preview.tsx
│   └── marketing-layout.tsx
├── app/
│   └── app-layout.tsx           <- Current layout
└── shared/
    └── market-pulse.tsx         <- Existing component
```

### 3. Data Integration Points
- **Live Stats**: Extend existing `/api/market-pulse` 
- **Tender Preview**: Read-only tender browsing
- **SEO Content**: Link to existing sector/supplier pages
- **Auth Flow**: Existing Neon Auth integration

### 4. 21st.dev Component Prompts

**Hero Section:**
"Create a modern hero section for a B2B SaaS platform. Include a main headline, sub-headline with live statistics, two CTA buttons (primary and secondary), and a subtle animated background. Use teal/mint color scheme. Professional, trustworthy feel for government contractors."

**Value Propositions:**
"Create three feature cards in a row layout. Each card should have an icon, headline, description, and subtle hover effects. Modern glassmorphism style with teal accent colors. Icons: search, target, trophy. Professional B2B design."

**Social Proof:**
"Create a social proof section with customer count, company logo grid, and success statistics. Clean, enterprise-grade design with subtle animations. Include testimonial quote area."

## Technical Requirements
- **SEO Optimized**: Meta tags, structured data
- **Performance**: Lazy loading, optimized assets
- **Responsive**: Mobile-first design
- **Accessibility**: ARIA labels, keyboard navigation
- **Analytics**: Conversion tracking on CTAs

## Success Metrics
- **Conversion Rate**: Sign-up clicks from marketing layout
- **Dwell Time**: Time spent on marketing content
- **SEO Performance**: Search rankings maintenance
- **User Flow**: Marketing → Sign up → App transition
